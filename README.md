# Backend Beginner

### Backend build using:
###### - Express.js
###### - PostgreSQL
###### - Node.js
###### - Test API & documentation with Postman

-------------

### npm Installation

#### - Using npm express
```bash
$ npm install express
```
#### - Using npm start
```bash
$ npm install start
```
#### - Using npm eslint
```bash
$ npm install i eslint
```
#### - Using npm body-parser
```bash
$ npm install i body-parser
```
#### - Using npm morgan
```bash
$ npm install i morgan
```
#### - Using npm pg for connect to PostgreSQL
```bash
$ npm install pg
```
#### - Using npm dotenv
```bash
$ npm install dotenv
```
